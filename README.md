# BMI Calculator Navigation App

This is a completed .NET MAUI BMI Calculator app with 3 pages and stack-based navigation.

## Features
- Weight slider (0-300 lbs)
- Height slider (0-100 inches)
- Gender selection by tapping male/female image
- BMI result page
- BMI category based on gender
- Personalized health recommendations page
- Stack-based navigation using NavigationPage

## Pages
1. Main Page - enter weight, height, and gender
2. BMI Result Page - shows BMI and category
3. Recommendations Page - shows health advice based on BMI and gender

## How to Run
1. Open the project in Visual Studio 2022 with the .NET MAUI workload installed.
2. Restore packages if prompted.
3. Select Windows Machine or an Android emulator.
4. Run the app.
